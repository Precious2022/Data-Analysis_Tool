# Data Analysis Tool

This project is a data analysis tool built using Python. It allows users to load CSV files, analyze the data, and generate summary statistics and visualizations.

## Features
- Load CSV files for analysis
- Compute summary statistics
- Generate and display visualizations such as closing prices and moving averages

# Installation
- Clone the repository:
- git clone <repository-url>
- cd DataAnalysisTool

- Set up Python virtual environment and install dependencies:

- python -m venv venv
- source venv/Scripts/activate   # for Windows
- source venv/bin/activate       # for MacOS/Linux
- pip install pandas matplotlib

- Run the analysis script:
- python analyze_data.py

# Project Structure
- analyze_data.py: Main script for loading and analyzing data.
- venv/: Virtual environment directory.
- data/: Directory containing sample CSV files.
- README.md: Project documentation (this file).

# Usage
- Place your CSV file in the data directory.
- Modify the analyze_data.py script to specify the correct file path.
- Run the script to generate summary statistics and plots.
